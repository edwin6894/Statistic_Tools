# Statistic_Tools
An All in One Educational-Research Purpose  Statistics tools 
This is a Tool developed by Edwin thomas using C# language and prompt engineering using Deepseek AI
The Software is set to provide users with the functions to perform statistical operations aiding high school/University students to their Statistic related problems which can be handy during assignment / project preparations, The Software has a user friendly GUI.

Upon Launching the software the user can see 3 Tabs at the menu bar
1. OPERATIONS : Provides a sub-menu that contains all Statistical operations
2. THEME: Allow the user to switch themes between dark/white/
3. ABOUT: A section that shows the Copyrights of the software and also About the creator

How to Use
------------------
1. Choose your desired operations from the OPERATIONS tab enter the required details in the textfield/ the field provided to enter the values and press ENTER button.
2. Upon typing in all values at once and pressing the ENTER key, a new CALCULATE Button appears , Click it to get your output answers.
3. Press the RESET Button to reset the software, and you can choose the next operation to perform
4. For Operations like GEOMETRIC MEAN(FREQUENCT DISTRIBUTION) the data will be displayed in a tabular format on the right-side of the software, a built-in EXPORT Button is provided that exports the data inside the table into .CSV File that the user can save in their PC.
